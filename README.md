# StudyLog

공부 기록을 남기고 학습 습관을 관리하기 위해 만든 간단한 웹 애플리케이션입니다.

## Demo
https://greengreenplant.github.io/studylog/

## Features
- 공부 기록 추가
- 공부 기록 삭제
- 총 공부시간 자동 계산
- LocalStorage 데이터 저장

## Tech Stack
HTML / CSS / JavaScript

## What I Learned
- DOM 조작을 통해 동적으로 리스트를 생성하는 방법
- 배열을 이용한 데이터 관리
- 이벤트 리스너를 통한 사용자 인터랙션 처리
- LocalStorage를 이용한 데이터 저장
- GitHub Pages를 이용한 웹 프로젝트 배포
